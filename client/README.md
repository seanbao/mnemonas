# MnemoNAS Flutter 客户端

[English](README.en.md) | 简体中文

本目录包含面向 Android、Linux 和 Windows 的 Flutter 客户端。Android 是首个可用平台目标；Linux 和 Windows runner 当前仅保留共享的跨平台工程边界，尚未完成对应原生主机上的构建和发布验证。

> [!WARNING]
> 客户端仍处于开发阶段，尚未发布任何可用版本或可分发安装包。当前源码和构建产物仅用于开发与验证。

## 当前开发范围

当前源码已接入：

- 服务器地址输入、服务状态验证和设备切换；
- 登录、安全会话存储、访问令牌刷新、强制修改密码、主动修改密码和退出登录；
- 设备概览及文件目录浏览；
- 新建文件夹、上传、下载、传输进度与取消、打开、重命名、移动、复制和基于服务端删除策略的删除确认；
- 客户端与服务端版本展示，以及 GitHub Issues 问题反馈入口。

以上能力仅表示当前源码中的实现范围，不表示已经形成可用客户端版本。

## 当前缺口

- 搜索和相册索引尚未接入；
- 版本历史、回收站管理、分享和管理功能尚未形成完整客户端流程；
- 后台传输、断点续传和进程重启后的传输恢复尚未完成；
- 当前界面文案以简体中文为主，完整多语言支持尚未完成；
- Linux 和 Windows 尚未完成原生构建与运行验证；
- Android 真实设备验收、升级验证、独立发布签名和正式发布产物尚未完成。

客户端允许本机或可信局域网使用 HTTP，并显示未加密警告；公网地址必须使用 HTTPS。

## 开发与反馈

构建、测试、安全、签名和设备验证要求见[开发指南](../docs/development.md)，英文说明见 [Development guide](../docs/development.en.md)。

缺陷、使用问题和功能建议可通过 [GitHub Issues](https://github.com/seanbao/mnemonas/issues) 提交。反馈应注明服务端 Git commit、客户端源码 commit、客户端版本、设备型号和操作系统版本。
